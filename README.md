# 🌩️ CLIMA-VIBE | Neon Edition

**Clima-Vibe** es una aplicación web moderna con estética Dark/Neon que consume la API de OpenWeather para dar información climática en tiempo real. 

---

## 🚀 Características
* 🌐 **Consultas Globales:** Obtén el clima de cualquier ciudad del mundo.
* 🎨 **Interfaz Neón:** Diseño optimizado para creadores digitales.
* 📱 **Responsive:** Totalmente funcional en dispositivos móviles y escritorio.
* ⚡ **Despliegue Continuo:** Alojado en Vercel para máxima velocidad.

## 🛠️ Tecnologías Usadas
* **HTML5 / CSS3** (Efectos de brillo neón)
* **JavaScript (Vanilla)** (Consumo de Fetch API)
* **OpenWeather API** (Datos meteorológicos)
* **Termux** (Entorno de desarrollo móvil)
